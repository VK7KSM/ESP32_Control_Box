协议逆向与数据分析阶段使用的 PC 端脚本工具。

## 文件说明

### `extract_logic_rows.py`

从逻辑分析仪截图中提取彩色协议字节行，辅助人工识别 TH-9800 串行帧结构。

**用途**：在逆向 TH-9800 通信协议时，将逻辑分析仪软件（如 PulseView/Saleae）导出的彩色截图批量处理，自动定位每个字节胶囊的像素坐标并读取颜色编码。

**依赖**：
```bash
pip install Pillow
```

**用法**：
```bash
python extract_logic_rows.py <截图目录> [--output <输出目录>]
```
